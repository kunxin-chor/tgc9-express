# tgc9-express

1. Install `yarn`, if your system doesn't have it yet.

```npm install -g yarn```

2. Install `nodemon` if your system doesn't have it yet.

```npm install -g nodemon```

3. Install all dependencie. You have to do this
for each folder.

```yarn install```

4. Dependencies:

* `yarn add express`
* `yarn add hbs`
* `yarn add handelbars-helpers`
* `yarn add wax-on`
